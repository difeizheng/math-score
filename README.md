## 🚀 快速开始

### 1. 安装后端依赖

```powershell
cd study\backend
pip install -r requirements.txt
```

### 2. 启动后端服务

**方式一：双击启动**
```
双击「启动后端.bat」
```

**方式二：命令行启动**
```powershell
cd study\backend
python main.py
```

后端将在 http://localhost:8000 启动

### 3. 打开前端页面

**方式一：双击启动**
```
双击「启动前端.bat」
```

**方式二：直接打开**
用浏览器打开 `study\frontend\index.html`

**方式三：本地服务器**
```powershell
cd study\frontend
python -m http.server 3000
```
然后访问 http://localhost:3000

## 📝 注意事项

1. **首次使用**先安装依赖（双击「安装依赖.bat」）
2. **数据导入**支持 Excel 拖拽上传
3. **学号是唯一标识**，确保准确
4. **数据存储在** `backend/scores.db`，注意备份
5. **打印模式**点击「🖨️ 打印模式」按钮
